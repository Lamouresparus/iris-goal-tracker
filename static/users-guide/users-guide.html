Overview Of The Iris Goals Tracker

    There are few sections on the goal tracker dashboard

  • users profile section
  • section to add goals and their description
  • section that displays added goals
  • section that gives users room to break down tasks to achieving each goals and displays the progress of each goals
  • section that gives summary of goal achievement. It displays
    • Number of All tracked goals
    • Number of All pending goals and
    • Number of All completed goals

Iris Goal Tracker

    How to Use the iris goal tracker

  • Create an account
  • Log in to access your goal tracker dashboard
  • Add Your Set goal and give it a description
  • On My Goals section, Click on a goal and see it display on the right hand side
  • Schedule tasks for set goals by updating your to do lists.
  • check all tasks completed to view your progress
  • on At A glance section, get the summary of:
    • total number of goals
    • total number of ongoing goals
    • total number completed goals
  • On the users profile setion, you can log out of your account for another.
